# e-plantShopping - Paradise Nursery

Welcome to **e-plantShopping**, a responsive React web application designed for **Paradise Nursery**—where green meets serenity. This application provides a seamless online shopping experience for plant lovers, allowing them to browse a wide variety of houseplants, view details, and manage their shopping cart.

## Features

- **Interactive Landing Page**: A beautiful welcome screen with an "About Us" section detailing our mission to provide high-quality plants.
- **Product Catalog**: A diverse listing of plants categorized by types (e.g., air-purifying, aromatic, easy-care) complete with descriptions, images, and prices.
- **Dynamic Shopping Cart**: Full cart management (add, remove, update quantities) with real-time updates of total items and cost.
- **Redux State Management**: Powered by `@reduxjs/toolkit` and `react-redux` for a predictable and centralized application state.
- **Sleek & Modern UI**: Built with React and customized CSS for a clean, intuitive, and responsive interface.

## Tech Stack

- **Frontend Framework**: React (Vite)
- **State Management**: Redux Toolkit & React Redux
- **Styling**: Vanilla CSS
- **Icons & Assets**: Custom visual assets and clean typography

## Getting Started

To run the application locally, follow these steps:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/<username>/e-plantShopping.git
   cd e-plantShopping
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Start the Development Server:**
   ```bash
   npm run dev
   ```

4. **Open in Browser:**
   Navigate to the local URL (usually `http://localhost:5173`) to view the application.

## Scripts

- `npm run dev` - Starts the Vite development server.
- `npm run build` - Builds the application for production.
- `npm run lint` - Runs ESLint to check for code quality issues.
- `npm run preview` - Previews the production build locally.
